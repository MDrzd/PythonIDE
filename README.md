#  Python IDE (Android)

A lightweight **Python IDE for Android** built using **Java + Chaquopy**, designed to run Python code directly on mobile with a simple IDE-like experience.

---

## Features

- Code Editor (Sora Editor)
- Python 3.11 runtime (Chaquopy)
- Run Python code instantly
- Material 3 UI design
- Fully offline (no internet required to run code)

---

## Tech Stack

- Java (Android)
- Chaquopy (Python in Android)
- Sora Editor (Code editor component)
- Material 3 Components
- AndroidX
- ExecutorService (background execution)
